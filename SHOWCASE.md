## 项目定位

李尔不是大型语言模型 API 的套壳，也不是只靠几个关键词返回固定句子的教程脚本。它是一套由人工知识库、规则处理器、混合检索、会话状态、确定性工具、角色人格和互动游戏共同组成的中文编程式对话系统。


截图的作用是展示部分真实运行片段，而不是穷举整个项目的能力。更完整的结构、复杂度和工程评估见 [TECHNICAL_REPORT.md](TECHNICAL_REPORT.md)。
